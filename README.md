# My Resume & Portfolio

A modern, responsive, and visually striking resume and portfolio website for MD Imtiyaz Hossain.

## 🚀 Project Overview
This project is a static portfolio and resume website, designed to showcase professional experience, education, skills, certifications, and achievements. It features a **Warm Coral Tech** theme with deep navy backgrounds and warm coral-orange accents for a contemporary, professional look.

- **Built with:** HTML5, CSS3 (custom & Bootstrap 5), and JavaScript (vanilla)
- **Theme:** Warm Coral Tech (Deep Navy, Dark Teal, Coral Orange, Soft White Text)
- **Responsive:** Fully mobile-friendly and cross-browser compatible
- **No frameworks:** Pure HTML/CSS/JS, Bootstrap only for layout
- **Pages:** Home, Resume (with Education, Experience, Skills, Certifications, Achievements)

## 🚀 Getting Started

### View Online
Simply open `index.html` in your web browser. No build process or dependencies required!

### Customize Content
1. **Update Personal Info:** Edit `index.html` for name, title, bio
2. **Modify Sections:** Update content in `pages/` folder
3. **Change Colors:** Edit CSS variables in `styles/custom.css`:
   ```css
   :root {
       --primary-color: #FF6B4A;      /* Coral Orange */
       --secondary-color: #F97316;    /* Soft Orange */
       --bg-color: #0B1D2A;           /* Deep Navy */
       --surface-color: #0F2A3D;      /* Dark Teal */
       --text-color: #F3F4F6;         /* Soft White */
   }
   ```
4. **Add Images:** Place profile and certificate images in `images/` folder
