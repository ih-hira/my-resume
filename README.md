# My Resume & Portfolio

A modern, responsive, and visually striking resume and portfolio website for MD Imtiyaz Hossain.

## 🚀 Project Overview
This project is a static portfolio and resume website, designed to showcase professional experience, education, skills, certifications, and achievements. It features a Midnight Tech (dark + electric accent) theme for a sleek, modern look.

- **Built with:** HTML5, CSS3 (custom & Bootstrap 5), and JavaScript (vanilla)
- **Theme:** Midnight Tech (Deep navy, Electric Cyan, Indigo, Light Text)
- **Responsive:** Fully mobile-friendly and cross-browser compatible
- **No frameworks:** Pure HTML/CSS/JS, Bootstrap only for layout
- **Pages:** Home, Resume (with Education, Experience, Skills, Certifications, Achievements)

## ✨ Features
- Modern, dark-themed UI with electric cyan and indigo accents
- Responsive layout for all devices
- Interactive navigation and smooth transitions
- Modular page structure for easy updates
- Custom scrollbar and hover effects
